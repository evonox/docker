Class TabReorderOperation

Method Object Pattern - used to process the drag-and-drop operation for the tab handle reordering

Events: onTabReordered - raised on successful tab reorder operation onCancelled - raised in case of the tab reorder operation is cancelled (e.g. by Escape key) onUndockRequest - raised when this operation transforms from tab reordering to undock operation
